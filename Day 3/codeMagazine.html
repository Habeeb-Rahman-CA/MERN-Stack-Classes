The Basic Language of the Web : HTML

Posted by Laura Jones on Monday, June 21st 2017

All modern websites and web applications are built using three fundamental technologies: HTML,CSS and JavaScript. These are the language of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTMl stands for Hyper Text Markup Language its a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consist of elements that describe different types of content: paragraph, link, heading, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML?

There are countless reason for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Worpress or Wix
  • To build web applications
